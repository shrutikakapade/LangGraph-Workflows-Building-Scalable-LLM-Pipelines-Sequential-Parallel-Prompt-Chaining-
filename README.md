<h2>About LangGraph</h2>
<p>
LangGraph is a framework used to design <b>stateful, multi-step LLM workflows</b>.  
It helps build applications where multiple steps, agents, or prompts interact in a structured flow.
</p>

<hr>

<h2>Workflows Covered</h2>

<h3>1️⃣ Simple Sequential Workflow</h3>
<p>Step-by-step execution where each task runs one after another.</p>

<h3>2️⃣ LLM-Based Sequential Workflow</h3>
<p>Each step uses an LLM for reasoning and content generation.</p>

<h3>3️⃣ Prompt Chaining Workflow</h3>
<p>Complex tasks are broken into smaller prompts where outputs feed the next step.</p>

<h3>4️⃣ Parallel Workflow</h3>
<p>Multiple tasks run simultaneously and their outputs are merged.</p>
